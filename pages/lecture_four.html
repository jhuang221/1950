CSS Backgrounds

Using images requires either the HTML img tag or the CSS background-image property.

When to use HTML image tag

  • for image infomation crucial to the page content
  • to allow the image to be indexed by search engines (SEO)
  • to allow using the alt attribute, for both SEO and screen readers

When to use CSS background-image

  • for images that contribute to the design of the site, rather than the content
  • for image effects that depend on CSS techniques (eg :hover pseudoclassing for image swap on mouseover)
  • repeating images
  • icons, textures, patterns

CSS supports several useful background related properties.

  • background-color:
    assign the background color
  • background-image: url()
    applies an image as the background.
  • background-repeat: repeat | repeat-x | repeat-y | no-repeat
    determines how a background image will repeat display. the default is repeat
    background-repeat:repeat;
    background-repeat:repeat-x;
    background-repeat:repeat-y;
    background-repeat:no-repeat;
  • background-size: <percentage> | <length>
    determines the size of a background image. you can use this to scale backgrounds to fit their parent containers
    background-size:100%;
    background-size:100% 50%;
    background-repeat:no-repeat;
    background-size:30px;
  • background-attachment: scroll | fixed
    determines if a background image will scroll with the parent element or the viewport
    background-attachment:fixed;
    background-attachment:scroll;

    background-attachment:fixed;
    background-size: cover;
    background-repeat:no-repeat;
    See an example of 'fixed' and 'cover'
